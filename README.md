# Classroom Sampler

AI-assisted systematic sampling tool for classroom assessment.

## What It Does
Helps enumerators generate a random list of students using 
systematic sampling — random start + fixed sampling interval.
Eliminates manual calculation errors and saves significant time.

## How to Use
1. Enter total number of students in class
2. Enter required sample size
3. App calculates random start and sampling interval
4. Generates the final list of selected students instantly
